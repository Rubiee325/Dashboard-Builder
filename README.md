# Dashboard Builder

A dynamic dashboard builder application where users can create customizable dashboards with charts, tables, and KPI widgets.

## Features
- Drag & Drop Dashboard
- Charts (Bar, Line, Pie, Area, Scatter)
- KPI Metrics
- Table Data View
- Real-time updates
- Dark/Light Mode
- PDF Export

## Tech Stack
- React.js
- Recharts
- Node.js
- Express.js
- MongoDB

## Run Project
```bash
npm install
npm start
